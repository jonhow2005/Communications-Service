Features Overview

This platform offers robust communication capabilities, including end-to-end encrypted (E2E) chats, non-encrypted chats, and real-time audio calls. Below are the key features users interact with:

Account Management

  • Unique IDs: Each user is assigned a unique ID to ensure individuality.
  • Online/Offline Status: Users can signal their availability by being online or offline.

Public Chats

  • Message Handling: Messages are sent to all participants and updated in real-time.
  • Hosting Costs: Users can host chats for a cryptocurrency fee.

Audio Calls

  • Real-Time Calls: Audio is transmitted in real time for seamless communication.
  • Priority Channels: Users can pay for priority access to improve call quality.

Conversations

  • Message Delivery: Messages are sent to the server and then delivered to the recipient upon availability.
  • Confirmation: Senders are notified once messages are received by the recipient.

Key Sending and Private Chats

  • Key Sharing: Users can exchange keys to enable E2E encryption.
  • Private Messaging: Messages are encrypted and decrypted using shared keys for secure communication.

Security and Privacy Commitment

This platform ensures a balance of flexibility and privacy for both secure and casual communication.
